## New in Excalidraw Automate
- `getPolyBool()` returns a [PolyBool](https://github.com/velipso/polybooljs) object
- sample mermaid code:
```js
ea = ExcalidrawAutomate();
ea.setView();
await ea.addMermaid(
  `flowchart TD
    A[Christmas] -->|Get money| B(Go shopping)
    B --> C{Let me think}
    C -->|One| D[Laptop]
    C -->|Two| E[iPhone]
    C -->|Three| F[fa:fa-car Car]`
);
ea.addElementsToView();
```
